Repository ini berisi source code untuk website "Furni", sebuah landing page sederhana untuk toko furniture. Project ini dibuat sebagai tugas mata kuliah Pemrograman Web untuk mempraktekkan dasar-dasar HTML, CSS, dan framework Bootstrap 5.

Tujuan utama dari pembuatan website ini adalah untuk belajar bagaimana membuat layout website yang rapi dan responsif (bisa menyesuaikan tampilan di HP maupun Laptop) tanpa menggunakan template jadi.
